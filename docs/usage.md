# Usage Guide

## Creating a New Package

```bash
pset create mypackage --author "Your Name" --email "your@email.com"
```
## Building a Package

```bash
cd mypackage
pset build
```

## Publishing to PyPI

```bash
pset publish
```

## Initializing an Existing Project

```bash
pset init
```

## Installing Dependencies

```bash
pset install requests
pset install pytest --dev
